A minimal boilerplate with Vite, Ant Design, Tailwind CSS, and TanStack Router.

## Quick Start

The required Node.js version is specified in the `engines` field of `package.json`

```sh
# 1. Install dependencies
npm ci

# 2. (Optional) Create local environment file
cp .env.example .env

# 3. Start the development server
npm run dev
```

## Available Scripts

-   **`dev`**: Starts the development server.
-   **`build`**: Builds the application for production.
-   **`test`**: Runs the test suite.
-   **`lint`**: Lints the codebase.
-   **`format`**: Formats all files with Prettier.
-   **`typecheck`**: Performs a TypeScript type check.
